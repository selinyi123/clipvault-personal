# 自用版策略

## 1. 自用版和商业版的区别

商业版要考虑：

```text
用户信任
隐私合规
上架审核
权限解释
多用户隔离
云服务稳定性
支付系统
客服支持
数据删除协议
```

自用版只需要考虑：

```text
我能不能稳定用
我能不能快速保存
我能不能双端同步
我能不能自动入 Obsidian
我能不能备份到 GitHub
我能不能通过输入法快速调用
我用起来是否舒服
```

## 2. 自用版可以简化

不做：

```text
账号系统
商业官网
支付系统
复杂云服务
多用户权限
iOS 第一阶段
上架审核
完整输入法竞争
皮肤商店
广告系统
```

保留：

```text
Windows Desktop
Android App
Android Keyboard Personal
Obsidian
GitHub 私有备份
SQLite
局域网/Tailscale 同步
个人词库
续词推荐
```

## 3. 自用版重点

```text
顺手
少打字
少切换
少选择
少重复输入
少手动归档
系统匹配度高
常用内容快速调用
```

## 4. 个人输入体验增强

最终目标：

```text
你刚输入两个字，它知道你大概率要说什么；
你刚复制一段内容，它知道你大概率要保存到哪里；
你刚打开输入法，它知道你最近最可能要粘贴什么；
你刚开始写项目规划，它知道你的常用结构、Prompt、命令和路径。
```
