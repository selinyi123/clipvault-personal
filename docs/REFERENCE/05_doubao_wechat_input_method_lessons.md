# 豆包输入法与微信输入法启发

## 1. 豆包输入法启发

豆包输入法给人的舒适感来自：

```text
AI 表达辅助
长输入能力
上下文联想
智能纠错
语音输入
常用工具前置
```

对 ClipVault 的启发：

```text
输入法不只是输入字符，
而是对当前文本提供下一步处理动作。
```

应用到 ClipVault：

| 内容类型 | 推荐动作 |
|---|---|
| Prompt | 结构化、版本化、归档、入库 |
| 代码 | 识别语言、解释、保存片段 |
| 错误日志 | 提取错误、生成排查清单 |
| URL | 摘要网页、保存链接、生成笔记 |
| 普通文本 | 摘要、改写、翻译、保存 |
| 命令 | 分类 Shell/Git/Docker、保存命令 |
| 密钥 | 隔离、隐藏预览、禁止同步 |

## 2. 微信输入法启发

微信输入法给人的舒适感来自：

```text
系统匹配度
续词推理
常用语
剪贴板
跨设备复制粘贴
词库同步
关键信息记忆
```

对 ClipVault 的启发：

```text
真正舒服的输入法不是功能多，
而是知道你下一步大概率想干什么。
```

应用到 ClipVault：

```text
个人词库
常用短语
常用 Prompt
常用命令
双端同步剪切板
输入法面板
前缀推荐
上下文续动作
```

## 3. 最终吸收

```text
豆包路线：AI 表达辅助与上下文处理
微信路线：系统融合、常用词记忆、续词推理

ClipVault Personal 应该融合：
输入体验舒适度
+ 个人记忆
+ 知识入库
+ 双端同步
```
